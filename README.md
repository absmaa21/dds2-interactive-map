## DDS2 Interactive Map

DDS2 Interactive Map is a web-based interactive map for **Drug Dealer Simulator 2**.
It provides an overview of important in-game locations and allows users to explore them through an intuitive map interface.

The project is designed as a lightweight frontend application focused on usability and fast interaction with map data.


### Features

* Interactive game world map
* Searchable and filterable locations
* Category toggles for different point types
* Clean and responsive UI
* Easily extendable location dataset

### Purpose

The goal of this project is to provide the DDS2 community with a simple and accessible tool
to explore the game world and locate important places more efficiently.

### Contributing

Contributions are welcome. Feel free to submit issues, improvements, or additional map data.


## How to run local

#### 1. Install Dependencies
```shell
bun install
```
or
```shell
npm install
```

#### 2. Start the local dev server
```shell
bun run dev
```
or
```shell
npm run dev
```
